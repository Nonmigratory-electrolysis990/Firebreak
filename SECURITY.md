# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in Firebreak, please report it responsibly.

**Email**: security@firebreak.dev

We will:
- Acknowledge your report within 48 hours
- Provide an estimated timeline for a fix
- Credit you in the release notes (unless you prefer anonymity)

**Do not** open a public issue for security vulnerabilities.

## Scope

This policy covers vulnerabilities in Firebreak itself — the MCP server, scanning engine, and associated tools. It does not cover vulnerabilities found in targets that Firebreak scans.

|---------|-----------|
